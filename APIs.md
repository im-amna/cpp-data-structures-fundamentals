
# STL APIs and Time Complexity

## Vector

APIs:

- push_back()
- pop_back()
- front()
- back()
- size()

Complexity:

- Access → O(1)
- Insert End → O(1)
- Search → O(n)

---

## Stack

APIs:

- push()
- pop()
- top()

Complexity:

- Push → O(1)
- Pop → O(1)
- Top → O(1)

---

## Queue

APIs:

- push()
- pop()
- front()
- back()

Complexity:

- Enqueue → O(1)
- Dequeue → O(1)

---

## Deque

APIs:

- push_front()
